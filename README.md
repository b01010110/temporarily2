### Запуск:

- Создать .env файл со следующим содержимым:

```
NODE_ENV=development
VUE_ROUTER_MODE=history
VUE_ROUTER_BASE=/
API_URL=https://jsonplaceholder.typicode.com
```

- Установить зависимости:

```
yarn install --frozen-lockfile
```

- Запуск:

```
yarn run dev
```
